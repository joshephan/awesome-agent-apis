## Contributing

Contributions welcome! Please read the [contribution guidelines](CONTRIBUTING.md) first.

**Criteria for inclusion:**
- ‚úÖ Publicly accessible API
- ‚úÖ JSON responses (or easily parseable format)
- ‚úÖ Stable and reliable
- ‚úÖ Good documentation
- ‚úÖ Preferably no auth or simple API key
- ‚úÖ Agent-friendly (predictable structure, no CAPTCHA)

**Not included:**
- ‚ùå APIs requiring OAuth with browser flow
- ‚ùå APIs with frequent breaking changes
- ‚ùå APIs requiring human verification (CAPTCHA)
- ‚ùå APIs with rate limits under 100 requests/day
